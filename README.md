# Reef-Gen

App for reef monitoring and coral health analysis

**Reef-Gen** is an AI-powered, crowdsourced platform for monitoring and restoring coral reefs.  
It allows divers, scientists, and communities to upload reef images/videos, get automated health diagnoses, and earn tokens through gamified conservation efforts.  

---

## Features

- Upload coral reef images and videos  
- AI-powered coral health diagnosis (mock or real AI model)  
- User management and authentication  
- Token-based gamification system to incentivize participation  
- Database tracking for scans, tokens, and metadata  
- Modular FastAPI backend with REST API endpoints  

---

## Tech Stack

- Python 3.x  
- FastAPI  
- SQLAlchemy + SQLite  
- Pydantic for data validation  
- Git Bash / Bash shell for development  
- Pillow / OpenCV for image processing  
- Docker-ready (optional)
